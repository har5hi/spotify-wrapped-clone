🎧 Spotify Wrapped Clone
A web application inspired by Spotify Wrapped that visualizes user listening insights with an interactive and modern UI.

Live Demo
👉 https://har5hi.github.io/spotify-wrapped-clone/

Features
🎵 Interactive Spotify Wrapped-style UI
📊 Data visualization using Chart.js
✨ Animated and responsive design
🎧 Interactive cards with hover effects
📸 Shareable wrapped summary (download as image)
🔄 Demo mode for showcasing UI without backend

🛠️ Tech Stack
HTML
CSS
JavaScript
Chart.js
Node.js (for local backend)
Git & GitHub

How to Run Locally

1. Clone the repository
git clone https://github.com/har5hi/spotify-wrapped-clone.git
cd spotify-wrapped-clone

2. Install dependencies
npm install

3. Start backend server
node server.js

4. Open frontend
Open `index.html` using Live Server or browser

Note
Spotify authentication works locally using backend support.
The deployed version runs in demo mode due to API restrictions.
