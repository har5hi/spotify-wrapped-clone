🎧 Spotify Wrapped Clone

A web application inspired by Spotify Wrapped that visualizes user listening insights with an interactive and modern UI.

🚀 Features
🎵 Spotify OAuth Login (local backend)
📊 Data visualization using Chart.js
✨ Animated and responsive UI
🎧 Interactive cards with hover effects
📸 Shareable wrapped summary (download as image)
🔄 Fallback demo mode for UI preview

🛠️ Tech Stack
HTML, CSS, JavaScript, Chart.js, Node.js (for local backend), Git & GitHub

How to Run Locally
1. Clone the repository

```bash
git clone https://github.com/har5hi/spotify-wrapped-clone.git
cd spotify-wrapped-clone
```
2. Install dependencies
```bash
npm install
``
```
3. Start backend server
```bash
node server.js
```
4. Open frontend
Open `index.html` using Live Server or browser
---

Note
Spotify API authentication works locally with backend support.
For deployed/demo versions, a fallback UI is shown due to API restrictions.
